<h2>how to use:</h2>
<ul>
  <li>pull out pegs</li>
  <li>move legs to the wanted position</li>
  <li>read insert pegs</li>
  <li>push the tiny flaps on the legs in</li>
  <li>position the fume extractor with the black filter facing away</li>
  <li>plug in</li>
</ul>
